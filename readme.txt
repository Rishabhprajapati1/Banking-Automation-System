📌 Project Overview-->
The Bank Automation System is a Windows-based desktop application designed to digitize and manage daily banking operations. It transitions traditional manual banking activities into an automated, user-friendly digital format. The system efficiently handles user account creation, transaction recording, and balance management.

🚀 Features-->
User Registration & Authentication: Secure login and registration modules ensuring that only authorized users with valid IDs and passwords can access the member's zone.
Account Management: Facilities for users to open new accounts and manage their personal details.
Transactions: Seamlessly perform core banking transactions, including:
->Depositing funds.
->Withdrawing funds.
->Transferring funds between accounts.
Account Dashboard: Users can check their current balance and view their transaction history.
Administrator Controls: A dedicated admin module to monitor site activity, view registered users, and edit information to maintain the service.

💻 Tech Stack-->
